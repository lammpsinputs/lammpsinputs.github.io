    "Here the surface tension of a vapour-liquid-vapour sytem is calculated using LAMMPS. The input file can be downloaded [here](../input/input.lammps). It has been prepared using the 23Jun2022 version of LAMMPS. The system looks like that:"
   ]

  {
   "cell_type": "markdown",

   "source": [
    "We anticipate that 4 processors will be used, and that the most sensitive way to cut the system is along the x and y dimensions:\n",
    "```\n",
    "processors 2 2 1\n",
    "```\n",
    "Always ensure that using more than one processor is advantageous in terms of computational time by running some short tests. For some small systems, using 1 processor is faster than uning 4."
   ]
  },
  {
   "cell_type": "markdown",
   "metadata": {},
   "source": [
    "Use periodic boundary conditions (`p`) in all 3 directions of space, the system of units `real`, and the atom style `full`:\n",
    "```\n",
    "boundary p p p\n",
    "units real\n",
    "atom_style full\n",
    "```"
   ]
  },
  {
   "cell_type": "markdown",
   "metadata": {},
   "source": [
    "Define style for bonds and angles:\n",
    "```\n",
    "bond_style harmonic\n",
    "angle_style harmonic\n",
    "```\n",
    "Those styles are irrelevants as we are using a rigid water model."
   ]
  },
  {
   "cell_type": "markdown",
   "metadata": {},
   "source": [
    "Define a pair style and a long-range solver:\n",
    "```\n",
    "pair_style lj/cut/tip4p/long 1 2 1 1 0.1546 12.0\n",
    "kspace_style pppm/tip4p 1.0e-5\n",
    "```\n",
    "The `lj/cut/tip4p/long` is similar to `lj/cut/coul/long` (Lennard-Jones interaction with a cut-off, and long range Coulomb interaction), but it has been made specifically to take care of the virtual site of the water molecule. For TIP4P/$\\epsilon$ water model, the OM distance is $0.1546\\,\\textrm{\\AA}$."
